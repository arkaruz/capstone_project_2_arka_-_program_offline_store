# capstone_project_2_arka_-_program_offline_store
Program ini digunakan untuk manajemen stok dan kasir sederhana berbasis python. Fitur utamanya adalah CRUD data produk (tambah/hapus), riwayat penghapusan, serta sistem belanja interaktif yang otomatis menghitung subtotal, memproses pembayaran, dan memperbarui jumlah stok secara real-time.
